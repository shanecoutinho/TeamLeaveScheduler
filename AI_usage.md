# AI Usage

## AI Tool Used

- ChatGPT (OpenAI)

---

## Most Useful Prompts

### Prompt 1

Help implement the business rules for:
- 30% team leave capacity
- overlapping leave requests
- weekends and public holidays

Why it helped:
It assisted in translating the business requirements into backend service logic.

---

### Prompt 2

Help build a React frontend that communicates with the ASP.NET Web API and allows users to:
- submit leave requests
- approve requests
- reject requests
- display approved and pending leave

Why it helped:
It accelerated development of the frontend while allowing me to focus on integrating the backend.

---

## One Case Where AI Was Incorrect

During frontend integration, AI suggested changes that introduced additional debugging steps around request payloads. I verified the implementation using browser developer tools and Swagger, then adjusted the solution to match the API requirements.

---

## How AI Was Used

AI was used as a development assistant for:
- explaining concepts
- generating boilerplate code
- debugging backend and frontend issues
- creating documentation

All generated code was reviewed, tested and modified where necessary before being included in the final solution.
